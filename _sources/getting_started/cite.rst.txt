Citing FAT Forensics
++++++++++++++++++++

Please remember to cite our package and user guide whenever you use it.

Citing the Package
==================

When referring to the package, its architecture or functionality, please use:

.. code:: latex

   @article{fat-forensics,
    title={{F}{A}{T}-{F}orensics: {A} {Python {T}oolbox for {A}lgorithmic
           {F}airness, {A}ccountability and {T}ransparency},
    author={Sokol, K. and Santos-Rodriguez, R. and Flach, P.},
    year={2019}
   }

Citing the Code
===============

When using the package itself, parts of the code or, in general, referring to
FAT Forensic's API, please use:

.. code:: latex

   @article{fat-forensics,
    title={{F}{A}{T}-{F}orensics: {A} {Python {T}oolbox for {A}lgorithmic
           {F}airness, {A}ccountability and {T}ransparency},
    author={Sokol, K. and Hepburn, A. and Poyiadzi, R. and Clifford, M.
            and Santos-Rodriguez, R. and Flach, P.},
    journal={JOOS},
    year={2019}
   }

Citing the User Guide
=====================

When referring to the :ref:`user_guide`, please use:

.. code:: latex

   @article{fat-forensics,
    title={{A}lgorithmic {A}pproach to {F}airness, {A}ccountability and
           {T}ransparency},
    author={Sokol, K. and Flach, P.},
    year={2019}
   }

Explainability Fact Sheets
--------------------------

When referring to the *Explainability Fact Sheets* used in the
:ref:`transparency part <user_guide_transparency>` of the :ref:`user_guide`,
please use:

.. code:: latex

   @article{fat-forensics,
    title={{E}xplainability {F}act {S}heets: {A} {F}ramework for {S}ystematic
           {A}ssessment of {E}xplainable {A}pproaches},
    author={Sokol, K. and Flach, P.},
    year={2019}
   }
