fatf.accountability.data.measures.sampling_bias_check¶

fatf.accountability.data.measures.sampling_bias_check(counts: List[int], threshold: float = 0.8) → bool[source]¶

Checks for a pairwise sampling bias based on the provided threshold.

Please see the documentation of fatf.accountability.data.measures.sampling_bias_grid_check function for a description of input parameters, errors and exceptions.

Returns
is_biasedboolean

True if any sub-group pair does not satisfy the sampling bias threshold, False otherwise.
